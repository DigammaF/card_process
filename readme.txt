
Python est nécessaire !

https://www.python.org/ (Downloads -> dernière version)

Mode d'emploi :

1) Lancer le programme une première fois pour créer les dossiers nécessaires
2) Placer les fichiers dans 'input'
3) Le résultat se trouvera dans 'output'

Il est possible de modifier 'settings.json' pour changer certains réglages. Pour remettre les réglages à défaut, il suffit de supprimer le fichier de configuration.

Note : lorsque vous configurez le type de fichier de sortie, veillez à selectionner un format qui supporte la transparence.
